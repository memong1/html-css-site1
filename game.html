개발한 객체지향 프로그래밍 언어입니다. 주로 Windows 애플리케이션 개발과 웹 개발에 사용되며, .NET 프레임워크를 기반으로 동작합니다. Java는 범용 프로그래밍

용도를 가지고 있습니다. C++은 성능에 초점을 둔 고급 프로그래밍, C#은 Windows 애플리케이션과 웹 개발, Java는 플랫폼 독립적인 개발, Lua는 경량 스크립팅 언어로 사용됩니다.
